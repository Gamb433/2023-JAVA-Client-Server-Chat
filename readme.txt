Najprej zaženite v enem terminalu
java ChatServer.java

Nato v vsaj treh drugih terminalih
java ChatClient.java 
in vpišite uporabniška imena ter si jih zapomnite.

JAVNA SPOROČILA:
- to so vsa sporočila, če ni uporabljen poseben regex za privatna spročila (glejte spodaj).

----------------------------
ZASEBNA SPOROČILA:
za njihovo uporabo je potreben naslednji regex, kjer so _ presledki:
private_{ime_prejemnika}_{sporočilo}

Primer zasebnega sporočila prejemniku "peter":

private peter To je testno sporočilo.
